Global Terrorism Dashboard

This dashboard visualizes terrorism data over time using three coordinated views.

Line Chart

This chart is an overview of the number of terrorist attacks per year. Use your mouse to click and drag over the chart to brush a time range. The Bar Chart view below updates based on the brushed selection.

Stacked Bar Chart

This chart shows attacks per region per year. You can zoom and pan to explore specific periods. When brushing from the overview, bars will animate in and out.

Advanced View - Streamgraph

This streamgraph displays the flow of different attack types over time. Each color band represents an attack type. Use the legend to identify each stream.
